# Animated Login Page


**Animated Login Page** is a modern, responsive and visually appealing login interface built with **ReactJS**. It features smooth animations, an interactive login form and a flexible layout that can display images, information or guidelines alongside the login section. The design emphasizes user experience with clean UI elements and fluid animations.


## Features

- Fully responsive three-section layout:
  - Left section: Image display
  - Center section: Informational content (e.g., healthcare guidelines)
  - Right section: Login form
- Animated gradient borders for login box
- Smooth input focus effects
- Mobile-friendly and responsive design
- Easy to integrate with any ReactJS project

## Installation & Setup

1. **Clone the repository**:

```bash
git clone https://github.com/iamx-ariful-islam/animated-login-page.git
cd animated-login-page
```

2. **Install dependencies using npm or npx:**

```bash
npm install
# or using npx
npx install
```

3. **Start the development server:**

```bash
npm start
```
4. **Open your browser and navigate to:**

```bash
http://localhost:3000
```

5. **The app will automatically reload if you make changes to the code.**
